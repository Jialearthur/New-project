# 发热门诊分诊指引 v1.0

## 1. 接诊范围

体温大于等于 37.3 摄氏度，或伴有咳嗽、咽痛、乏力等急性呼吸道症状的患者，应优先引导至发热门诊预检分诊。

患者在候诊和就诊期间应规范佩戴口罩，并按照院感要求与其他患者保持间距。

## 2. 分诊要点

分诊护士应记录体温、主要症状、起病时间、流行病学接触史及基础疾病情况。

对于老年患者、孕妇以及免疫功能低下患者，应在登记后尽快安排医师评估。

## 3. 危急值处理

若患者血氧饱和度低于 93%，或出现明显呼吸窘迫、意识模糊、持续胸闷等情况，应立即启动绿色通道处理。

仅有低热且生命体征平稳的患者，可按普通发热门诊流程进行后续检查和处置。
