# 常用抗菌药物用药提醒 v2.0

## 1. 使用原则

抗菌药物应在明确或高度怀疑细菌感染时使用，不得将其作为普通病毒性感冒的常规治疗方案。

医师开具抗菌药物前，应结合感染部位、严重程度、既往药物过敏史及当地耐药情况综合判断。

## 2. 过敏史核对

对于拟使用青霉素类或头孢菌素类药物的患者，接诊医师应先核对既往药物过敏史。

若患者既往存在明确严重速发型过敏反应史，应及时向上级医师报告并重新评估用药方案。

## 3. 用药观察

首次使用静脉抗菌药物后，应密切观察患者是否出现皮疹、呼吸困难、胸闷或血压下降等异常反应。

一旦怀疑发生药物过敏或输液反应，应立即停止给药，并按照科室急救流程处理。
