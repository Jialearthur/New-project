# Project A 演示问题模板

答辩前建议准备 10 到 20 份企业制度文档，并围绕下面问题做演示。

## 推荐演示问题

1. 员工年假天数如何按照工龄计算？
2. 试用期员工是否可以申请事假？
3. 员工迟到几次会触发绩效扣分？
4. 加班调休的申请流程是什么？
5. 病假需要提交哪些证明材料？
6. 员工报销流程分为哪几个步骤？
7. 出差住宿标准上限是多少？
8. 请假审批需要经过哪些角色？
9. 公司的考勤打卡时间要求是什么？
10. 如果制度中没有提到陪产假，系统会怎么回答？

## 演示顺序建议

1. 先问一个制度中有明确条款的问题
2. 再问一个需要跨段落理解的问题
3. 最后问一个知识库中没有答案的问题，展示系统拒答能力

## 评分亮点

- 文档上传后能自动建库
- 问答结果有引用来源
- 没有依据时不会编造答案
- 后台能查看问答日志
